🏭 Pengaturan Perusahaan
📜 Kode Google Apps Script

📋 Langkah Setup (ikuti urutan)

  1. Buat Google Spreadsheet baru → copy ID dari URL
  2. Buat folder Google Drive → copy ID dari URL
  3. Di Spreadsheet → Extensions → Apps Script
  4. Hapus kode default, paste kode di bawah
  5. Ganti SPREADSHEET_ID & FOLDER_ID
  6. Deploy → New Deployment → Web App
  7. Execute as: Me | Access: Anyone
  8. Copy URL deployment → paste di field Script URL → Simpan
